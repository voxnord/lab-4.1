Задание 6: Разработка системы для онлайн-магазина
Описание: Создайте систему для управления онлайн-магазином, включая управление товарами, заказами и клиентами.
Реализовать:
!1. Класс Product с методами для управления информацией о товаре (цена, описание, количество).
!2. Класс Customer для управления информацией о клиентах и их заказах.
!3. Класс Order, который содержит список товаров и информацию о заказе.
!4. Реализуйте метод для добавления товаров в заказ и оформления заказа.
!5. Метод для отслеживания статуса заказа (например, в обработке, отправлен, доставлен).
!6. Класс ShoppingCart для управления текущими товарами в корзине клиента.
!7. Реализуйте метод для генерации отчетов о продажах по товарам и клиентам.
!8. Класс Category для организации товаров по категориям.
!9. Метод для обновления информации о товарах (цена, количество).
!10. Интерфейс для управления заказами и товарами через веб-приложение или GUI.
